<Steps>

1.  Install the [system package for Elixir](https://archlinux.org/packages/extra/any/elixir/).

2.  Install the [Erlang Core](https://archlinux.org/packages/extra/x86_64/erlang-core/).
    You can instead choose to install the complete [Erlang](https://archlinux.org/packages/extra/x86_64/erlang/) package, which `erlang-core` is a part of.

</Steps>
